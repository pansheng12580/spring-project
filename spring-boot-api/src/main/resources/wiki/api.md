后台接口API wiki
=====


查询首页
--------

* 请求类型	`GET`
* 请求URL	`http://127.0.0.1:9002/countries`
* 请求参数说明

|   字段名	|       描述	      |	 可选      |
|:---------:|:-----------:|:-----------:|
|pageNum	|当前页码（默认1）			|  可选  |
|pageSize	|当前页码item总数（默认1）		|  可选  |


* 返回参数样例


	{
	    "retcode": "00000000",
	    "retmsg": "查询热门标的成功！",
	    "list": []
	}





常见错误码
-------
	
`00000000` 成功
`99999999` 系统错误，请联系管理员
